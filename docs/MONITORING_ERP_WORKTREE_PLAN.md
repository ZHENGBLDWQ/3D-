# LayerTrace 监控型经营系统 Worktree 执行方案

## 产品边界

- Bambu Studio 是唯一的模型准备、切片、参数调试、打印下发和设备操作端。
- LayerTrace 不切片、不下发、不暂停或停止打印，只采集打印机与 AMS 数据。
- LayerTrace 负责订单、实时打印监控、耗材库存、成本与定价、经营分析和系统设置。
- 无法自动识别的产品、订单或实体耗材卷必须进入待认领/待绑定，不允许猜测扣库。

## 最终一级模块

1. 首页工作台
2. 订单中心
3. 实时打印监控
4. 耗材库存
5. 成本与定价
6. 经营分析
7. 系统设置

耗材库存内部固定分为：

- 库存管理：未拆封库存、采购入库、在途、领用、调拨、退库、盘点、损耗、报废、低库存、补货建议和库存流水。
- 使用中：在机耗材、已开封周转卷、AMS/外置料盘、主/辅助挤出机、实时预计用量和任务结算。

## 权威库存规则

1. 采购收货生成采购批次和唯一实体卷。
2. 领用时未拆封仓库卷数立即减少，实体卷转为使用中；组织总资产不减少。
3. 打印开始只预留预计耗材，不立即正式扣减。
4. 打印结束后按照材料用途和实体卷生成不可变消耗流水。
5. 退库后的开封卷进入已开封周转，不重新计入未拆封库存。
6. 称重校准优先级高于切片估算和打印遥测，差异必须生成调整流水。

## 打印材料分类

每次打印允许多条材料行，至少支持：

- model
- support
- support_interface
- purge
- wipe_tower
- brim
- calibration
- unknown

每条材料行记录工具头、供料来源、AMS 单元/槽位、实体卷、预计克重、结算克重、称重克重和成本。X2D 必须支持主/辅助工具头与外置料盘。

## Worktree 与依赖

### 10. `agent/phase-10-material-domain`

唯一 schema owner。新增目录、采购批次、实体卷、位置、移动流水、供料位置、绑定、打印会话、多材料用量行和称重校准模型。负责迁移与兼容投影，不建设页面。

### 11A. `agent/phase-11-inventory-v2`

依赖阶段 10。建设“库存管理 / 使用中”双区、未拆封库存、开封周转、扫码领用、退库、在机余量和分类流水。

### 11B. `agent/phase-11-bambu-monitor`

依赖阶段 10。统一旧 Agent 与 gateway 事件，提供只读 `BambuMonitorAdapter`、打印会话幂等、断网补传、AMS/外置/工具头快照和待绑定。

### 11C. `agent/phase-11-business-hubs`

可与阶段 10 并行，只新增订单、成本、分析和设置 Hub，复用现有 API，不修改 schema、`app/page.tsx` 或 `app/globals.css`。

### 12. `agent/phase-12-slice-metadata`

依赖阶段 10。建设已切片 3MF 监控目录和 G-code 分类解析器，按 plate、filament、toolhead 和 feature 形成预计用量；不调用打印控制。

### 13. `agent/phase-13-cost-settlement`

依赖 11A、11B、12。完成打印结束分类结算、失败按层累计、单次打印成本、单件良品成本和订单成本回写。

### 14. `agent/phase-14-information-architecture`

最后合并。唯一允许集中修改 `app/page.tsx`、共享侧栏和全局导航的 worktree；接入七个一级模块，隐藏旧切片、下发、预检、排产和控制入口，但保留旧 URL 兼容。

## 冲突所有权

- `db/schema.ts`、`db/ensure-schema.ts`、Drizzle journal：阶段 10 独占。
- `app/page.tsx`、`app/globals.css`、共享导航：阶段 14 独占。
- Agent/gateway 与打印事件契约：阶段 11B 独占。
- 库存 API 与库存页面：阶段 11A 独占。

## 验收门槛

- Studio 发起任务后 30 秒内出现在实时监控。
- 相同文件重打、Agent 重启和断网补传不漏记、不重复。
- 未绑定实体卷时只告警，不自动猜测扣库。
- 未拆封、开封周转、在机和已消耗四种状态账实可追溯。
- 支持 AMS、AMS Lite、AMS HT、外置料盘及主/辅助工具头。
- 主体、支撑、支撑界面、擦拭塔、冲刷和校准可以分别核算。
- 所有重量修改都有不可变流水和组织隔离。
- A1、P2S、X2D 各至少一台真实设备连续验证两周后才标记生产稳定。
